# Anritsu - File Export Formats

The usermanual describes the ability to export a number of different file formats

https://dl.cdn-anritsu.com/en-us/test-measurement/files/Manuals/Users-Guide/10580-00475H.pdf

- Anritsu DGZ (IQ Capture): Select if exporting to an Anritsu DGZ file.

- Anritsu WVD: Select when exporting to an Anritsu WVD file (Vector Signal 
Generator format). Anritsu WCAP (IQ Capture) Select if exporting to an Anritsu WCAP file.

-  Anritsu.vsg: Select if exporting to an Anritsu.vsg file.
 Binary: Select if exporting to single raw data file that has IQ values. 
Set the bit format as 8 bit, 16 bit, 24 bit, 32 bit Float, or 32 bit Integer. 
 Set the Endianness as (LE) Little Endian or (BE) Big Endian.

- Text: Set the Text delimiter that appears between the I and Q values as Comma, 
Tab, or Semicolon. Auto is not an option in the Export File Type menu.

- SpectroX XDat File: Select if exporting to a SpectroX Xdat file. The .xdat file 
format is the file format required for analyses using the X-Com SpectroX software; 
.xdat files are for use in the SpectroX software. xdat files support 8, 10, 16, and 32 
bits. 24-bit files will not be converted.

-  Wave File: Select if exporting to a Wave file.
